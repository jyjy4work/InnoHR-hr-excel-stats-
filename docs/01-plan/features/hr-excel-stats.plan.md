## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | HR 담당자가 엑셀 파일에서 수동으로 인원 현황·입퇴사 통계를 집계하는 데 많은 시간을 소비하며, 오류와 일관성 부재가 발생한다. |
| **Solution** | 표준 HR 엑셀 파일을 업로드하면 자동으로 파싱하여 인원 현황·입퇴사·이직·유사 카테고리 통계를 시각화하고 결과를 내보낼 수 있는 로컬 Streamlit 앱을 제공한다. |
| **Functional UX Effect** | 드래그 앤 드롭 업로드 → 자동 분석 → 대시보드 확인 → 엑셀/CSV 다운로드까지 클릭 몇 번으로 완료, HR 리포트 작성 시간을 대폭 단축한다. |
| **Core Value** | 반복적인 HR 데이터 집계를 자동화하여 HR 담당자가 데이터 해석과 의사결정에 집중할 수 있도록 한다. |

## 1. 배경 및 목적

### 1.1 문제 정의
- HR 담당자는 매월/분기별로 인원 현황 보고서를 수작업으로 작성
- 엑셀 파일에서 피벗·필터·수식 작업에 반복 시간 소요
- 다수의 파일 버전으로 인한 데이터 일관성 부재
- 시각화 차트 수작업 생성으로 추가 시간 발생

### 1.2 목표
- HR 엑셀 파일 업로드 → 자동 통계 산출 → 시각화 대시보드 제공
- 결과를 엑셀/CSV로 다운로드 가능
- 비개발자(HR 담당자)도 쉽게 사용 가능한 UI

## 2. 요구사항

### 2.1 기능 요구사항

#### FR-01: 엑셀 파일 업로드
- `.xlsx`, `.xls`, `.csv` 파일 지원
- 파일 크기 제한: 50MB 이하
- 업로드 후 컬럼 자동 감지 및 미리보기 제공
- 표준 포맷에 맞는 컬럼 자동 매핑

#### FR-02: 인원 현황 통계
- 전체 재직 인원수
- 부서별 인원 분포 (막대그래프)
- 직급별 인원 분포 (막대/원형 차트)
- 성별 비율 (원형 차트)
- 연령대별 분포 (히스토그램)
- 고용 형태별 분포 (정규직/계약직/파견 등)
- 근속연수 분포

#### FR-03: 입퇴사/이직 통계
- 월별/분기별 입사자 수 추이 (라인 차트)
- 월별/분기별 퇴사자 수 추이 (라인 차트)
- 이직률 계산 (퇴사자/평균재직자 × 100)
- 퇴사 사유별 분류 (있을 경우)
- 부서별 이직률 비교
- 평균 재직기간

#### FR-04: 추천 추가 통계 카테고리
- **채용 현황**: 채용 채널별 합격률, 채용 소요일
- **교육/역량**: 교육 이수율, 교육 시간 분포
- **조직 구조**: 스팬 오브 컨트롤(관리자 1명당 부하 직원 수), 팀 규모 분포

#### FR-05: 대시보드 뷰
- 모든 주요 통계를 한 화면에서 확인하는 종합 대시보드
- 기간 필터 (날짜 범위 선택)
- 부서 필터 (다중 선택)

#### FR-06: 결과 내보내기
- 엑셀(.xlsx) 다운로드 (통계 시트 포함)
- CSV 다운로드
- 차트 이미지 저장 (PNG)

### 2.2 비기능 요구사항
- 로컬 실행 (인터넷 연결 불필요)
- Python 3.8+ 환경
- 엑셀 업로드 후 30초 내 결과 표시
- 한국어 UI
- 민감 데이터 미저장 (세션 내 처리 후 폐기)

## 5. 사용자 플로우

```
[앱 실행] → [파일 업로드 화면]
     ↓
[엑셀 파일 드래그 앤 드롭 or 클릭 업로드]
     ↓
[컬럼 자동 감지 + 미리보기 (3행)]
     ↓ (컬럼 매핑 확인/수정)
[분석 시작 버튼 클릭]
     ↓
[대시보드 화면]
  ├─ [인원 현황 탭]
  ├─ [입퇴사/이직 탭]
  ├─ [추가 통계 탭]
  └─ [전체 대시보드 탭]
     ↓
[필터 적용 (기간/부서)]
     ↓
[엑셀/CSV/이미지 다운로드]
```

## 8. 구현 범위 (In/Out of Scope)

### In Scope
- Python Streamlit 로컬 앱
- 인원 현황, 입퇴사/이직 통계
- 채용 현황, 교육/역량, 조직 구조 통계 (데이터 있을 경우)
- 인터랙티브 차트 (Plotly)
- 대시보드 + 탭 네비게이션
- 기간/부서 필터
- 엑셀/CSV/이미지 내보내기
- 컬럼 매핑 UI

### Out of Scope
- 클라우드 서버 배포
- 데이터베이스 저장
- 사용자 인증/권한 관리
- 실시간 데이터 연동 (ERP/HRIS 연동)
- 모바일 앱
